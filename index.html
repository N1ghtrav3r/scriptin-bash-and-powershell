Theorie: read opties

Wat doet read?

Met read vraag je de gebruiker om iets in te typen. Wat de gebruiker typt wordt opgeslagen in een variabele.

OptieBetekenisVoorbeeld
-p "tekst"Toont een prompt (tekst) voor de invoer. Staat voor prompt.read -p "Naam? " naam
-sMaakt de invoer onzichtbaar (verborgen). Staat voor silent. Gebruik je voor wachtwoorden.read -s -p "Wachtwoord: " ww
-t 10Geeft de gebruiker 10 seconden om iets in te typen. Daarna gaat het script verder.read -t 10 -p "Snel! " keuze
-n 4Stopt automatisch na 4 tekens invoer.read -n 4 -p "PIN: " pin
-rBehandelt backslash als gewone tekst (raw mode).read -r -p "Pad: " pad
Na een read -s staat de cursor nog op dezelfde lijn. Zet altijd een lege echo erna voor een nieuwe lijn!
read -p "Naam? " naam          # toont "Naam? " en wacht
read -s -p "Wachtwoord: " ww  # invoer onzichtbaar
echo                          # nieuwe lijn na -s
Bash

Theorie: if / elif / else

Wat is een if-structuur?

Een if-structuur laat je script een beslissing nemen. "Als dit waar is, doe dan dat. Anders doe iets anders."

OnderdeelBetekenis
if [ ... ]; thenStart de controle. Als de voorwaarde waar is, wordt alles tot elif/else/fi uitgevoerd.
elif [ ... ]; thenOptioneel. Een tweede controle als de eerste niet waar was. Kan je meerdere keren gebruiken.
elseOptioneel. Wordt uitgevoerd als geen enkele voorwaarde hierboven waar was.
fiSluit het if-blok af. Verplicht! (fi = if achterstevoren)
Spaties in [ ] zijn verplicht! [ "$x" = "y" ] is correct. ["$x"="y"] geeft een fout.
if [ "$score" -ge 50 ]; then
    echo "Geslaagd"
elif [ "$score" -ge 30 ]; then
    echo "Bijna"
else
    echo "Niet geslaagd"
fi

Twee voorwaarden tegelijk (-a en -o)

Met -a (AND) controleer je of twee dingen tegelijk waar zijn. Met -o (OR) of minstens één waar is.

if [ "$user" = "admin" -a "$ww" = "1234" ]; then
    echo "Beide correct!"
fi
Bash

Theorie: while & for loops

while loop

Een while loop herhaalt een blok code zolang een voorwaarde waar is. Zodra de voorwaarde niet meer waar is, stopt de loop.

OnderdeelBetekenis
while [ voorwaarde ]; doZolang de voorwaarde waar is, blijf herhalen.
while true; doEeuwige loop — blijft herhalen totdat jij hem stopt met break.
breakStopt de loop onmiddellijk, ook al is de voorwaarde nog waar.
doneSluit het while-blok af. Verplicht!
teller=0
while [ $teller -lt 5 ]; do
    echo "Teller: $teller"
    (( teller++ ))
done

# Eeuwige loop met break
while true; do
    read -p "Doorgaan? (j/n) " keuze
    [ "$keuze" = "n" ] && break
done

for loop

Een for loop herhaalt een blok code voor elk item in een lijst, of een bepaald aantal keren.

VormWanneer gebruiken
for item in lijst; doAls je een vaste lijst hebt om door te lopen.
for (( i=1; i<=5; i++ )); doAls je een teller nodig hebt (zoals in C).
# Over een lijst
for fruit in appel peer banaan; do
    echo "$fruit"
done

# Met teller
for (( i=1; i<=5; i++ )); do
    echo "Ronde $i"
done
Bash

Theorie: Functies

Wat is een functie?

Een functie is een stukje code met een naam dat je meerdere keren kan aanroepen. Zo vermijd je herhaling in je script.

OnderdeelBetekenis
naam() { ... }Definieert de functie. De code erin wordt pas uitgevoerd als je de functie aanroept.
naam "argument"Roept de functie aan. Argumenten geef je mee na de naam.
$1, $2, ...Binnen de functie: de meegegeven argumenten.
return 0 / return 1Geeft een exitcode terug (0 = succes, 1 = fout). Geen echte returnwaarde!
Functies moet je definiëren voor je ze aanroept in het script.
begroet() {
    echo "Hallo, $1!"   # $1 = eerste argument
}
begroet "Gunther"       # output: Hallo, Gunther!
Bash

Theorie: Arrays

Wat is een array?

Een array is een variabele die meerdere waarden tegelijk kan bevatten. Elke waarde heeft een positienummer (index), startend vanaf 0.

SyntaxBetekenis
naam=("a" "b" "c")Array aanmaken met 3 elementen.
${naam[0]}Eerste element opvragen (index 0).
${naam[@]}Alle elementen opvragen.
${#naam[@]}Aantal elementen in de array.
naam+=("d")Element toevoegen aan de array.
kleuren=("rood" "groen" "blauw")
echo "${kleuren[0]}"    # rood
echo "${kleuren[@]}"    # rood groen blauw
echo "${#kleuren[@]}"   # 3
kleuren+=("geel")       # geel toevoegen

for kleur in "${kleuren[@]}"; do
    echo "$kleur"
done

Praktijkvoorbeelden

Voorbeeld 1: Wachtwoord bevestigen

Vraag de gebruiker een wachtwoord in te voeren en bevestigen. Blijf vragen zolang ze niet overeenkomen.

#!/bin/bash
# Gebruiker om wachtwoord vragen + bevestiging

while true; do
    read -s -p "Wachtwoord: " ww1
    echo
    read -s -p "Bevestig wachtwoord: " ww2
    echo

    if [ "$ww1" = "$ww2" ]; then
        echo "Wachtwoord ingesteld!"
        break
    else
        echo "Wachtwoorden komen niet overeen, probeer opnieuw."
    fi
done

Stap voor stap uitgelegd:

LijnUitleg
#!/bin/bashShebang: gebruik Bash als interpreter.
while true; doStart een eeuwige loop. Blijft herhalen totdat break uitgevoerd wordt.
read -s -p "..." ww1Vraag invoer (-p toont tekst, -s maakt het onzichtbaar). Sla op in ww1.
echoNieuwe lijn na de verborgen invoer, anders staat alles op één lijn.
if [ "$ww1" = "$ww2" ]; thenVergelijk de twee strings. = voor strings, niet -eq (dat is voor getallen).
breakStop de while loop zodra het wachtwoord correct bevestigd is.
fiSluit het if-blok af.
doneSluit het while-blok af. Loop herstart als break niet bereikt werd.

Voorbeeld 2: Login simulatie

Simuleer een login met een vaste gebruikersnaam en wachtwoord. Beide moeten correct zijn.

#!/bin/bash
# Vaste logingegevens bovenaan definiëren
correcte_gebruikersnaam="admin"
correcte_wachtwoord="1234"

while true; do
    read -p "Gebruikersnaam: " gebruiker
    read -s -p "Wachtwoord: " wachtwoord
    echo

    if [ "$gebruiker" = "$correcte_gebruikersnaam" -a "$wachtwoord" = "$correcte_wachtwoord" ]; then
        echo "Welkom!"
        break
    else
        echo "Ongeldige inloggegevens, probeer opnieuw."
    fi
done

Stap voor stap uitgelegd:

LijnUitleg
correcte_gebruikersnaam="admin"Sla de correcte waarden op als variabele bovenaan het script.
read -p "..." gebruikerGebruikersnaam is zichtbaar (-p zonder -s).
read -s -p "..." wachtwoordWachtwoord is verborgen (-s).
-aAND operator: beide voorwaarden moeten tegelijk waar zijn.
"$gebruiker" = "$correcte_gebruikersnaam"Vergelijk ingevoerde waarde met de vaste variabele.

Voorbeeld 3: Getal raden

Raad een vast getal. Het script geeft aan of je gok te hoog of te laag was, en telt je pogingen.

#!/bin/bash
correct_getal="42"
teller=0

while true; do
    read -p "Geef een getal in: " getal
    (( teller++ ))

    if [ "$getal" -eq "$correct_getal" ]; then
        echo "Juist! Je had $teller pogingen nodig."
        break
    elif [ "$getal" -gt "$correct_getal" ]; then
        echo "Te hoog!"
    else
        echo "Te laag!"
    fi
done

Stap voor stap uitgelegd:

LijnUitleg
correct_getal="42"Het te raden getal, bovenaan als variabele zodat je het makkelijk kan aanpassen.
teller=0Start de teller op 0. Elke poging verhogen we die met 1.
while true; doEeuwige loop — blijft vragen totdat de gebruiker het juiste getal raadt.
read -p "..." getalVraag een getal aan de gebruiker en sla het op in de variabele getal.
(( teller++ ))Verhoog de teller met 1 bij elke poging. Staat vóór de if zodat ook de eerste poging meegeteld wordt.
-eqVergelijkt twee getallen op gelijkheid. Gebruik altijd -eq voor getallen, niet = (dat is voor strings).
-gtGroter dan (greater than). Als getal groter is dan correct_getal → Te hoog.
elseAls geen van de bovenstaande voorwaarden waar is, moet het getal te laag zijn.
echo "... $teller ..."Toont de variabele teller in de boodschap bij een juist antwoord.
breakStopt de while loop zodra het juiste getal geraden is.
